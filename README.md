# DTunnel VPN

DTunnel VPN é um projeto desenvolvido em Kotlin para fornecer um serviço de VPN seguro e eficiente.

## Funcionalidades

- Criptografia de ponta a ponta
- Alta velocidade de conexão
- Interface amigável
- Suporte a múltiplos protocolos

## Instalação

Para instalar e executar o DTunnel VPN, siga os passos abaixo:

1. Clone o repositório:
    ```bash
    git clone https://github.com/DTunnel0/DTunnel_VPN_SRC.git
    ```

2. Navegue até o diretório do projeto:
    ```bash
    cd DTunnel_VPN_SRC
    ```

3. Instale as dependências:
    ```bash
    ./gradlew build
    ```

4. Execute a aplicação:
    ```bash
    ./gradlew run
    ```

## Uso

Após a instalação, você pode configurar e iniciar sua conexão VPN através da interface de usuário ou via linha de comando.
